    # 图书分类法

    图书分类法又叫图书分类词表，是按照图书的内容、形式、体裁和读者用途等，在一定的哲学思想指导下，运用知识分类的原理，采用逻辑方法，将所有学科的图书按其学科内容分成几大类，每一大类下分许多小类，每一小类下再分子小类。最后，每一种书都可以分到某一个类目下，每一个类目都有一个类号。

    分类词表是层次结构的类号和类目的集合。
